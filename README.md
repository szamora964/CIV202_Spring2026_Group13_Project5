# CIV202_Spring2026_Group13_Project5

This GitHub repo is my starter package for Project #5 in CIVE 202: Civil Engineering Analysis II. For this project, I decided to study riverine flooding across six states that are all connected by the Mississippi River corridor: Minnesota, Wisconsin, Iowa, Illinois, Missouri, and Arkansas. I chose these states because they are close to each other and share similar river systems, which makes it easier to look for regional patterns in flood risk.
The first step in this project is choosing the hazard and study area. My hazard is riverine flooding, and my study area is these six neighboring states. I think this is a strong choice because rivers and floodplains naturally connect the region, so the comparison between states should make sense.
The second step is gathering the data. I need two datasets for this analysis. The first is the FEMA National Risk Index (NRI) census tract dataset, and the second is the CDC/ATSDR Social Vulnerability Index (SVI) U.S. census tract dataset. Since the project requires census tract-level data, both of these sources fit the assignment well. Because I am comparing multiple states, I need to use the U.S. SVI tract dataset instead of separate state SVI files.
The third step is organizing the files in my repo. After downloading the datasets, I will place them in the data/raw/ folder. I should rename the FEMA file to nri.csv and the SVI file to svi.csv. This will make it easier for the notebook and script to read the files correctly.
The fourth step is running the analysis. This repo includes a few files to help with that. The notebooks/project5_initial_results.ipynb file is the Jupyter notebook version of the project, and scripts/project5_analysis.py is the Python script version. There is also a report_draft.md file that I can use as a starting point for my written explanation, along with a requirements.txt file for the Python packages I may need. Once the data is in the correct folder, I can open the notebook and run the cells to start getting results.
The fifth step is deciding what results to show. For the initial submission, I only need one clear result, so I do not need to finish the entire project yet. A good starting point would be a summary table and one draft figure, such as a boxplot or bar chart comparing riverine flood risk across the six states. That would meet the project requirement and also give me something clear to present as evidence of progress.
The main question I am trying to answer is: How does riverine flood risk differ across six neighboring Mississippi River corridor states, and how does that risk relate to tract population and social vulnerability? One trend I want to test is whether states farther south in the corridor have higher typical flood risk or more extreme high-risk tracts, while more populated areas may have a larger number of people exposed even if the average tract risk is not the highest.
One important thing to remember is that the FEMA NRI dataset may use different column names depending on the version of the file. Because of that, the code is set up to look for likely riverine flood columns automatically. If it does not find the correct columns, I may have to manually update the variables called MANUAL_HAZARD_RISK_COL and MANUAL_HAZARD_EAL_COL in the notebook or script.
